CONFIGURATION WORDPRESS :
=========================

1. PAGES PARENTES :
   - Creer d'abord la page "OEuvres" (slug: oeuvres)
   - Puis creer les 5 sous-pages (emergence, presence, revelation, interiorite, equilibre)
     en selectionnant "OEuvres" comme page parente
   - Cela creera automatiquement les URLs /oeuvres/emergence/, etc.

2. PAGE D'ACCUEIL :
   - Aller dans Reglages > Lecture
   - Choisir "Une page statique"
   - Selectionner la page "Accueil" comme page d'accueil

3. PERMALIENS :
   - Aller dans Reglages > Permaliens
   - Choisir "Nom de l'article" (/%postname%/)
   - Cela garantit des URLs propres comme /oeuvres/, /bio/, etc.
